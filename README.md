# 🏠 House Price Predictor - Erbil

ئەم پرۆژەیە مۆدێلێکی فێربوونی ئامێرە بۆ پێشبینیکردنی نرخی فرۆشتن و کرێی خانووەکان لە شاری هەولێر.

---

## 📊 تواناییەکانی مۆدێل

| جۆر | R² | ڕونکردنەوە |
|-----|----|-------------|
| فرۆشتن | 0.897 | 89.7% دروستی |
| کرێ | 0.771 | 77.1% دروستی |

- ڕووکاری سادەی Streamlit
- هەڵەی مامناوەندی فرۆشتن: $24,245
- Biasی گشتی: -0.7%
